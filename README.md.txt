# Indian Stock Analysis System 📈

This project is a live Indian stock market analysis system built using Python.

It calculates:

- SMA (Simple Moving Average)
- EMA (Exponential Moving Average)

and generates:

- BUY Signal
- SELL Signal
- HOLD Signal

based on SMA & EMA crossover strategy.

---

# Features 🚀

✅ Live Stock Data using Yahoo Finance  
✅ Supports Indian Stocks  
✅ SMA & EMA Calculation  
✅ BUY / SELL Trend Detection  
✅ Graph Visualization  
✅ CSV Data Download  
✅ Streamlit Web Application  

---
